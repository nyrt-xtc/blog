小天才root教程

2026-04-22 root root

小天才root教程最新版

前言

  • 手表root后可以安装第三方，magisk等高级操作。本教程会介绍root前准备和root的过程。
  • root后可能会导致云控和禁言，无法发布朋友圈，使用权益卡等云端功能。
  • 少年版的文件和数据线等都是和普通版一样的。
  • root文件下载,
    有问题请前往QQ群：876331156。
  • root使用的一键工具可以用:alltoolbox或者小天才全系列root工具。

一、准备

  • 确认能否root

    1. 确认是否支持root的方法为:
      1. 打开手表设置
      2. 划到底部打开我的设备或类似按钮
      3. 点击本机信息
      4. 查找是否有安卓版本信息
    2. 没有安卓版本，说明手表是RTOS系统，安装第三方需要其他的方法。
    3. 有的型号root方法并没有公开或研究出来，如z11。

  • 在手机端小天才app关闭系统自动更新和应用自动更新

  • 数据备份

    • 手表root后本地数据会丢失，不包括好友，运动，权益卡等云端数据。
      • 判断方法：手机端app能看到的数据都是云端数据。
    • 备份方法：
      • 照片表情包可以发到手机端保存或存到云盘。
      • 微聊记录无法备份。

  • 购买数据线

    • 官方数据线无法传输数据，需要购买4点线或卡槽线。
    • 判断方法：
      • 如果充电口有四个点或安卓版本为4.4.4和7.1的手表，购买四点线。
      • 如果只有两个点，需要卡槽线。
      • 现在买的安卓8.1及以上手表大部分都要卡槽线。
    • 购买方法：在闲鱼购买六点线，在拼多多等平台购买四点线（需要对应型号）。
    • 卡槽数据线使用方法请询问卖家。

二、root

root环节请拔出电话卡，断网。手表用数据线连接电脑，全程不要断开。

安卓4.4.4版本

  1. 打开拨号，输入*#0769651#*，打开adb。

  2. 电脑端安装驱动：adb驱动，紫光展锐驱动包。

  3. root环节

    • D3使用D3root工具箱一键root，选择第二种模式然后保留数据。
    • 其他4.4.4手表，使用工具选择一键root。

安卓7.1.1版本

  1. 打开拨号，输入*#0769651#*，打开adb。

  2. 电脑端安装驱动：adb驱动，小天才电话手表USB驱动。

  3. root环节

    • 使用工具选择一键root。

安卓8.1版本

  1. 电脑端安装驱动：adb驱动，小天才电话手表USB驱动。

  2. root环节

    1. 开启adb:
      1. 使用一键工具选择一键root。
      2. 短接手表，如果工具提示降级请先超级恢复到对应版本。
      3. 工具会刷入misc，进入此页面后点击重启-重启到正常模式。
      4. 开机后打开拨号，输入*#0769651#*，打开adb。
    2. 继续root，会再次进入qmmi，不要点重启！。
    3. 出现提示时,请根据指示操作手表。

Z10

  • z10需要手动使用qfil或其他工具刷入分区，没能力的建议找人远程。

  • 驱动：adb驱动，小天才电话手表USB驱动。

  • 文件在123的z10root260116.zip中，下载可能会很久。

具体过程

  1. 刷281固件

  2. 刷完后刷
    abl_a，
    vbmeta_a ，
    vbmeta_system_a，
    boot，
    super，
    misc(刷misc.bin)。

  3. 重启进入fatbootd后选reboot to bootloader，按一次电源键进bl

  4. 执行CMD命令：fastboot boot recovery.img

  5. 请在Dm-Verity.zip的文件目录下打开CMD后执行：

adb shell
twrp mount vendor
twrp mount system
twrp format data
twrp mount data
twrp sideload
adb sideload Dm-Verity.zip
(这一部卡在63%是正常的，等待一分钟后即可手动强制重启)

  • 温馨提示：刷281的时候，请更换QFIL的fh loader，刷改版固件时，请重装QFIL
